# Wand Hardware

This folder groups the physical wand implementations and related notes.

## Current Layout

- [current_led_wand/](current_led_wand/)
  the final physical wand direction used for the project demo.
- [legacy_esp32/](legacy_esp32/)
  the earlier ESP32-based concept kept for historical reference.

This keeps the repo honest about the project outcome:

- the final system used a simple 3D-printed LED wand
- the ESP32 concept remains documented, but it was not the final integrated
  wand used in the demo
